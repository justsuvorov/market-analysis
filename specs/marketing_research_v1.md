# Спецификация: Маркетинговое исследование коттеджного комплекса

## Status
`Draft`

## Purpose
Провести маркетинговое исследование для оценки перспективности коттеджного комплекса
с гостиничным управлением в Ленинградской области (50–100 км от Санкт-Петербурга).

Цель: снизить инвестиционные риски, обоснованно выбрать концепцию и локацию,
сформировать маркетинговую модель проекта.

Источник требований: `Коттеджный_комплекс_ТЗ_на_маркетинговое_исследование_Финал.docx`

---

## Исходные данные проекта

| Параметр | Значение |
|---|---|
| Рабочее название | Коттеджный комплекс с гостиничным управлением |
| Регион | Ленинградская область |
| Приоритетная локация | Сосновый Бор (спот для серфинга, природная привлекательность) |
| Альтернативные локации | Приморский р-н, Курортный р-н, побережье Ладоги |
| Радиус от СПб | 50–100 км |

### Приоритетный формат
- Коттеджные дома для аренды и/или продажи через управляющую компанию (УК)
- УК гостиничного типа: ресепшн, клининг, бронирование, техобслуживание
- Общая инфраструктура: детская площадка, банный комплекс, зоны отдыха, парковка
- **Двойная монетизация**: проживание владельца + арендный доход через УК

### Альтернативные сценарии
1. Коттеджный посёлок (ИЖС, продажа земли с коммуникациями)
2. Загородный клуб (таунхаусы, теннисные корты, банный комплекс)
3. Эко-отель / база отдыха (эко-ферма, гостевой дом, бар)
4. Смешанные сценарии

## Inputs

**Модель:** `ResearchInput`

| Поле | Тип | Описание |
|---|---|---|
| `region` | `str` | Регион поиска (Ленинградская область) |
| `center_city` | `str` | Центр радиуса (Санкт-Петербург) |
| `radius_km` | `tuple[int, int]` | Диапазон радиуса в км (50, 100) |
| `priority_location` | `str` | Приоритетная локация (Сосновый Бор) |
| `alternative_locations` | `list[str]` | Список альтернатив |
| `competitor_count` | `int` | Целевое кол-во конкурентов для анализа (10–15) |
| `failed_cases_count` | `int` | Мин. кол-во неудачных кейсов (2) |
| `audience_segments` | `list[str]` | Сегменты ЦА (инвесторы, покупатели, арендаторы) |

## Outputs

**Модель:** `ResearchReport`

| Поле | Тип | Описание |
|---|---|---|
| `market_overview` | `MarketOverview` | Обзор рынка: объём, динамика 3–5 лет, сезонность, тренды |
| `competitors` | `list[CompetitorProfile]` | 10–15 объектов-конкурентов |
| `failed_cases` | `list[CompetitorProfile]` | ≥2 неудачных кейса с разбором ошибок |
| `locations` | `list[LocationScore]` | Скоринг-матрица 3–4 локаций |
| `audience_segments` | `list[AudienceSegment]` | Портреты 3 сегментов ЦА |
| `concept_variants` | `list[ConceptVariant]` | 2–3 варианта концепции с УТП |
| `expert_interview_guide` | `ExpertInterviewGuide` | Вопросы для 4 глубинных интервью |
| `swot` | `SWOTAnalysis` | SWOT-анализ проекта |
| `conclusions` | `str` | Итоговые выводы и рекомендации |

## ML подход

### ResearcherAgent → AnalystAgent
- **CatBoost (классификация):** сегментация конкурентов по формату и ценовому сегменту
- **CatBoost (регрессия):** скоринг локаций по взвешенным критериям
- **GLM (statsmodels):** интерпретация весов факторов локации для бизнес-отчёта (коэффициенты = объяснение для инвестора)

### Признаки для скоринга локаций
| Признак | Тип | Бизнес-смысл |
|---|---|---|
| `distance_spb_min` | float | Время в пути от СПб (мин) |
| `land_price_rub_sotka` | float | Стоимость земли (руб/сотка) |
| `competition_density` | int | Кол-во конкурентов в радиусе 10 км |
| `water_proximity_km` | float | Расстояние до воды (км) |
| `unique_attraction` | bool | Наличие уникального аттрактора (серфинг, etc.) |
| `infrastructure_score` | int | Балл инфраструктуры района (1–10) |
| `legal_restrictions` | bool | Наличие правовых ограничений |
